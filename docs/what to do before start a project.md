- 代码静态检测，阿里巴巴规范，sonar检测等
- 系统架构图，模块划分(领域划分)
- 公共功能：统一日志处理，统一配置中心，公共定时器，session获取，导入，导出工具
- 文档记录形式，文档记录约定/规定
- 接口多版本支持思考
- 二维码形式（先调研，尽量简单易扩展）
- 用户系统统一，避免多套用户系统，如果多个用户之间信息共享会引发更多的问题
